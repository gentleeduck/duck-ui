## API Reference

### **Toggle Properties**

* **`variant`** (`'default' | 'outline'`, optional): Defines the visual style of the toggle.
* **`size`** (`'default' | 'sm' | 'lg'`, optional): Sets the size of the toggle button.
* **`...props`** (`Omit<React.HTMLProps<HTMLInputElement>, 'size'> & VariantProps<typeof toggleVariants>`): Additional props are forwarded to the underlying Radix `Toggle` component.
